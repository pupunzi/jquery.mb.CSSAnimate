jquery.mb.CSSAnimate demo

Animate using CSS3 transition instead of the default jQuery.animate method.
It provides a fallback to the default animate method if CSS3 is not supported by the browser.
Get the code

Translate

animate({ x: 50, y: 30 });

Translate (css)

css({ x: 50, y: 30 });

Rotate

animate({ rotate: 45 });

Rotate (css)

css({ rotate: 45 });

Scale

animate({ scale: 2 });

Scale (css)

css({ scale: 2 });

skew

animate({ skew: -50 });

skew (css)

css({ skew: -50 });

skew - rotate

animate({ skew: -50 }).animate({rotate:-120});
